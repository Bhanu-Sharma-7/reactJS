# useState Hook - ReactJS

useState hook React mein "State" manage karne ke kaam aata hai. Simple shabdon mein: Agar aapko apni screen par kuchh aisa data dikhana hai jo user ke action se change ho sake toh wahan useState use hota hai

## Syntax of useState

const [data, setData] = useState(initialValue)

- data: Current value
- setData: Wo function jo value ko update karega
- initialValue: Shuruat mein value kya hogi
