# ReactJS kya hai?

React ek javascript library hai jise Facebook ne banaya hai. Iska kaam hai fast aur sundar user interfaces banana.

# React kyun seekhein?

React ki sabse badi khoobi hai SPA - Single Page Application.

Aapne Facebook ya instagram use kiya hoga. Jab aap kisi post ko like karte hain toh pura page refresh nahi hota hai sirf woh chota sa "Like" button update hota hai. React yahi kaam karta hai woh sirf usi hisse ko update karta hai jise badalne ki zaroorat hai.
